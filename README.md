# powerbi-sales-dashboard

📊 Business Sales Performance Analytics
📌 Project Overview

This project is about analyzing business sales data to understand how a company is performing in different areas such as products, categories, and regions.

The main purpose of this analysis is to find meaningful insights that can help businesses make better decisions.
Using the Superstore dataset, I studied sales trends, profit performance, and customer demand patterns.

This project helped me understand how data analysts work with real business data to identify growth opportunities and improve overall business performance.

🛠️ Tools Used

• Power BI – used to create interactive dashboards and visualizations
• Microsoft Excel – used for data cleaning and basic data preparation
• Dataset – Superstore Sales Dataset

These tools helped me convert raw sales data into clear and understandable business insights.

🔎 Key Insights

• Some products had high sales but generated low profit, which shows margin issues
• Sales performance was stronger in certain regions compared to others
• The Technology category contributed a major share of total revenue
• Sales trends showed growth during specific time periods
• Some product categories had steady demand but inconsistent profitability

These findings can help businesses focus more on profitable areas and improve weaker segments.

📊 Dashboard Features

• Overview of total sales and profit with key performance indicators (KPIs)
• Comparison of sales performance across different regions
• Analysis of category-wise and product-wise performance
• Visualization of sales trends over time
• Interactive filters to explore data easily

The dashboard is designed to be simple, professional, and business-focused so that stakeholders can quickly understand performance and make informed decisions.
